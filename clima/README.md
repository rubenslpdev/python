# Clima & Clima Saudação

2 simple scripts to show the weather. 

## Clima

It is a more complete version showing the weather for the next few days. It finds your location using your IP and pull the information by API.

## Clima Saudação

It works exactly like **CLima** but simpler. Its function is to serve as a *welcome message* to display you user name, location and weather when opening terminal.
